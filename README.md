## 🎯 About the Repository

This repository is designed as a **complete Python programming learning workspace** for students and beginners. It combines:

- 📘 **Theory notes** for core Python concepts
- 📓 **Jupyter notebooks** for interactive learning
- 🧪 **Lab experiment notebooks** for practical sessions
- 🧠 **Question bank and class tests** for assessment preparation
- 📦 **Local package demo** using `dummycalc`
- 🚀 **FastAPI mini-project** for understanding basic backend API development

It can be used for classroom teaching, lab delivery, self-study, revision, assignments, and beginner-level project practice.

## 🚀 Mini Projects & Demos

### 1. `dummycalc` — Local Python Package Demo

The [`dummycalc/`](dummycalc/) folder demonstrates how a local Python package can be structured and imported.

It includes examples for:

- Basic arithmetic operations
- Advanced mathematical utilities
- Reports and helper functions
- System-related helper tools
- Importing package functions into scripts and notebooks

Example usage:

```python
from dummycalc import add, quick_report

print(add(20, 22))
print(quick_report(9, 7))
```

Run the root demo:

```bash
python main.py
```

### 2. `dummy_fastapi_project` — Student API Demo

The [`dummy_fastapi_project/`](dummy_fastapi_project/) folder contains a beginner-friendly FastAPI application using SQLite and SQLAlchemy.

Core API features:

| Feature | Description |
|---|---|
| Create Student | Add a new student record |
| List Students | View all students |
| Get Student | Retrieve a student by ID |
| Update Student | Modify student details |
| Delete Student | Remove a student record |
| SQLite Database | Stores records in `students.db` |
| Swagger UI | Auto-generated API testing interface |

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/vibhug0077/Python_Programing.git
cd Python_Programing
```

### 2. Create a virtual environment

#### Windows

```bash
python -m venv .venv
.venv\Scripts\activate
```

#### macOS / Linux

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Install useful packages

```bash
pip install jupyter notebook ipykernel fastapi uvicorn sqlalchemy "pydantic[email]"
```

### 4. Start Jupyter Notebook

```bash
jupyter notebook
```

Open any `.ipynb` file and run cells step by step.

## 🌐 Running the FastAPI Demo

Go to the FastAPI project folder:

```bash
cd dummy_fastapi_project
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the API server:

```bash
uvicorn main:app --reload
```

Open in browser:

```text
http://127.0.0.1:8000
http://127.0.0.1:8000/docs
```

Use `/docs` to test API endpoints using the built-in Swagger UI.

## 🧹 Recommended Best Practices

To keep the repository clean and professional, consider the following improvements:

### Add a `.gitignore`

```gitignore
__pycache__/
*.pyc
.venv/
.ipynb_checkpoints/
*.db
.env
```

### Suggested naming convention

Use consistent file naming such as:

```text
chapter_01_python_basics.ipynb
chapter_02_operators_input_output.ipynb
lab_01_python_installation_basics.ipynb
lab_02_input_operators.ipynb
```

## 🤝 Contribution Guidelines

For students and contributors:

1. Create a new branch for your work.
2. Keep file names clear and meaningful.
3. Add comments where logic may not be obvious.
4. Do not commit temporary files such as `__pycache__`, `.venv`, or notebook checkpoints.
5. Test scripts before pushing.
6. Keep notebooks clean by restarting the kernel and running all cells before submission.
